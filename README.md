🚀 Backend API for **FixMyResume**, an ATS-optimized resume enhancer that matches resumes with job descriptions.

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot
- **Authentication:** JWT-based authentication
- **Database:** MongoDB
